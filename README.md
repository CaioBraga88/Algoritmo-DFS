# Labirinto DFS - Teste de Memória de Movimentos

## Visão Geral

Labirinto DFS é um jogo desafiador onde o jogador precisa memorizar e replicar exatamente o caminho percorrido por um objeto controlado pelo algoritmo Depth-First Search (DFS). O objetivo é percorrer o labirinto seguindo os mesmos movimentos do objeto DFS, sem cometer nenhum erro. Se o jogador errar um movimento sequer, a fase é reiniciada, testando sua memória e precisão.

## Tecnologias Utilizadas

* Unity
* C#

## Como Executar

Para jogar o Labirinto DFS, siga as instruções abaixo:

### Pré-requisitos

* Unity Editor instalado na versão **6000.0.46f1**.

### Passos

1.  Clone este repositório para o seu computador:
    ```bash
    git clone https://github.com/CaioBraga88/Algoritmo-DFS.git
    ```
2.  Abra o Unity Editor.
3.  Na janela de abertura do Unity, clique em "Open" e selecione a pasta onde você clonou o repositório do projeto "Labirinto".
4.  Dentro do Unity Editor, navegue até a pasta de "Scenes" no painel "Project".
5.  Abra a cena principal do jogo ("Mapa1.unity").
6.  Clique no botão "Play" (o ícone de triângulo) no topo do Unity Editor para iniciar o jogo.

## Controles

O jogador pode controlar seu personagem utilizando as seguintes teclas:

* **Esquerda:** Tecla Left Arrow ou A
* **Direita:** Tecla Right Arrow ou D
* **Cima:** Tecla Up Arrow ou W
* **Baixo:** Tecla Down Arrow ou S

## Funcionalidades Principais

* **Mecânica de Memorização:** O jogador precisa memorizar a sequência de movimentos do objeto DFS.
* **Desafio de Precisão:** Cada movimento do jogador deve corresponder exatamente ao movimento do DFS.
* **Reinício de Fase:** Qualquer erro no movimento do jogador resulta no reinício imediato da fase.
* **Progressão de Fases:** O objetivo principal é avançar pelas fases seguindo corretamente os movimentos do DFS.
